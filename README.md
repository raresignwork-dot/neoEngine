This is for learning DirectX and contains errors and bugs, so please be careful when using it.

+ Overview
This project was developed using Visual Studio (latest version) and Direct3D11.
The engine is built on the following technologies:

- Direct3D11
- Direct2D
- DirectWrite
- DXGI

Asset importing currently uses Assimp, but a custom asset management system may be implemented in the future.

+ Status
This project is currently incomplete and under development.
The Assimp implementation is not fully finished and may produce errors.  
If you wish to run the project, please ensure that all dependencies are properly configured.

+ Purpose
This project was started primarily as a learning project to study the basic architecture of game engine development.
Because of this, many parts of the system are experimental and incomplete.
