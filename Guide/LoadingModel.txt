## Blender のマテリアル構造がどう流れてくるか
Blender では 1 つの Mesh に複数のマテリアルスロットを設定できます。

- Mesh  
  - Material Slot 0 → マテリアル A  
  - Material Slot 1 → マテリアル B  
  - Material Slot 2 → マテリアル C  

glTF にエクスポートすると：

- Mesh は複数の Primitive に分割される  
- 各 Primitive が materialIndex を持つ  
- Assimp の aiMesh も materialIndex を持つ  

つまり：

> **Blender のマテリアル割り当ては、Assimp の aiMesh::mMaterialIndex にそのまま反映される。**

あなたの Mesh.materialIndex は正しくこの値を受け取っています。

## Scene 全体に複数モデルがある場合
SceneContext → SceneManager → Scene → ModelBase → Node → Mesh  
という階層構造になっているので、

- Scene に複数の ModelBase  
- ModelBase に複数の Mesh  
- Mesh に materialIndex  
- ModelBase.materials に複数の Material  

という構造が自然に成立します。

# 🎉 完全版 ShaderManager（コピペ可）

## ShaderManager.h

```cpp
#pragma once
#include <string>
#include <unordered_map>
#include <memory>
#include "Shaders/Shader.h"

class ShaderManager
{
public:
    ShaderManager(ID3D11Device* device)
        : device_(device)
    {}

    // シェーダーを読み込む（初回のみ）
    std::shared_ptr<Shader> Load(const std::string& name,
                                 const std::wstring& file,
                                 const std::string& vs,
                                 const std::string& ps)
    {
        auto shader = std::make_shared<Shader>();
        if (!shader->Load(device_, file, vs, ps))
            return nullptr;

        shaders_[name] = shader;
        return shader;
    }

    // 名前で取得（既にロード済みなら即返す）
    std::shared_ptr<Shader> Get(const std::string& name)
    {
        auto it = shaders_.find(name);
        if (it != shaders_.end())
            return it->second;

        return nullptr;
    }

private:
    ID3D11Device* device_;
    std::unordered_map<std::string, std::shared_ptr<Shader>> shaders_;
};
```

## 統合シェーダーの全体像（コピペ可）

```hlsl
// =========================
// 定数バッファ
// =========================

// b0 : PerObject
cbuffer PerObjectCB : register(b0)
{
    float4x4 world;
};

// b1 : PerFrame
cbuffer PerFrameCB : register(b1)
{
    float4x4 view;
    float4x4 projection;
    float3   lightDir;      // 簡易ライト
    float    pad0;
    float3   cameraPos;
    float    pad1;
};

// b2 : Bones
cbuffer BoneCB : register(b2)
{
    float4x4 boneMatrices[128];
};

// b3 : Material
cbuffer MaterialCB : register(b3)
{
    float4 baseColor;       // RGBA
    float3 diffuseColor;    // Phong 用
    float  shininess;       // Phong 用

    float3 specularColor;   // Phong 用
    float  metallic;        // PBR 用

    float  roughness;       // PBR 用
    int   usePBR;           // 0 = Phong, 1 = PBR
    int   useTexture;       // 0 = 色のみ, 1 = テクスチャ使用
    int   pad2;
};

// テクスチャ
Texture2D baseColorTex : register(t0);
SamplerState samLinear : register(s0);

// =========================
// 頂点シェーダ
// =========================

struct VS_INPUT
{
    float3 position    : POSITION;
    float3 normal      : NORMAL;
    float2 uv          : TEXCOORD;

    uint4  boneIndices : BLENDINDICES;
    float4 boneWeights : BLENDWEIGHT;
};

struct VS_OUTPUT
{
    float4 Pos        : SV_POSITION;
    float3 WorldPos   : TEXCOORD;
    float3 WorldNormal: TEXCOORD1;
    float2 uv         : TEXCOORD2;
};

VS_OUTPUT VSMain(VS_INPUT input)
{
    VS_OUTPUT output;

    // ---- スキニング ----
    float4 skinnedPos = float4(input.position, 1.0f);
    float3 skinnedNormal = input.normal;

    if (any(input.boneWeights != 0))
    {
        float4 p = float4(0, 0, 0, 0);
        float3 n = float3(0, 0, 0);

        [unroll]
        for (int i = 0; i < 4; i++)
        {
            uint  idx = input.boneIndices[i];
            float w   = input.boneWeights[i];

            float4x4 m = boneMatrices[idx];

            p += mul(float4(input.position, 1.0f), m) * w;
            n += mul(float4(input.normal, 0.0f), m).xyz * w;
        }

        skinnedPos    = p;
        skinnedNormal = normalize(n);
    }

    // ---- 通常の変換 ----
    float4 worldPos = mul(skinnedPos, world);
    float4 viewPos  = mul(worldPos, view);
    float4 projPos  = mul(viewPos, projection);

    float3 worldNormal = normalize(mul(float4(skinnedNormal, 0.0f), world).xyz);

    output.Pos         = projPos;
    output.WorldPos    = worldPos.xyz;
    output.WorldNormal = worldNormal;
    output.uv          = input.uv;

    return output;
}

// =========================
// ピクセルシェーダ
// =========================

float3 BRDF_Phong(float3 N, float3 L, float3 V, float3 diffuseCol, float3 specCol, float shininess)
{
    float3 R = reflect(-L, N);
    float  NdotL = saturate(dot(N, L));
    float  RdotV = saturate(dot(R, V));

    float3 diffuse  = diffuseCol * NdotL;
    float3 specular = specCol * pow(RdotV, shininess);

    return diffuse + specular;
}

float3 BRDF_PBR(float3 N, float3 L, float3 V, float3 baseCol, float metallic, float roughness)
{
    // 超簡易 PBR（本格的にするなら GGX 等を追加）
    float3 H = normalize(L + V);

    float NdotL = saturate(dot(N, L));
    float NdotV = saturate(dot(N, V));
    float NdotH = saturate(dot(N, H));

    float3 F0 = lerp(float3(0.04, 0.04, 0.04), baseCol, metallic);
    float3 F  = F0 + (1.0 - F0) * pow(1.0 - NdotV, 5.0);

    float  rough2 = roughness * roughness;
    float  D = rough2 / max(0.001, (3.14159 * pow((NdotH * NdotH * (rough2 - 1.0) + 1.0), 2.0)));
    float  k = (roughness + 1.0) * (roughness + 1.0) / 8.0;
    float  Gv = NdotV / (NdotV * (1.0 - k) + k);
    float  Gl = NdotL / (NdotL * (1.0 - k) + k);
    float  G  = Gv * Gl;

    float3 specular = (D * F * G) / max(0.001, 4.0 * NdotL * NdotV);
    float3 kd = (1.0 - F) * (1.0 - metallic);
    float3 diffuse = kd * baseCol / 3.14159;

    return (diffuse + specular) * NdotL;
}

float4 PSMain(VS_OUTPUT input) : SV_Target
{
    float3 N = normalize(input.WorldNormal);
    float3 L = normalize(-lightDir);
    float3 V = normalize(cameraPos - input.WorldPos);

    float4 albedo = baseColor;
    if (useTexture)
    {
        albedo *= baseColorTex.Sample(samLinear, input.uv);
    }

    float3 color;

    if (usePBR == 1)
    {
        color = BRDF_PBR(N, L, V, albedo.rgb, metallic, roughness);
    }
    else
    {
        color = BRDF_Phong(N, L, V, diffuseColor, specularColor, shininess);
    }

    return float4(color, albedo.a);
}
```

## どういう思想になっているか

### 1. 頂点シェーダ

- 今のスキニングロジックをそのまま拡張  
- StaticModel：`boneWeights == 0` → スキニングスキップ  
- SkinnedModel：boneMatrices を使って位置と法線を変換  
- 出力は：
  - `WorldPos`（ライティング用）  
  - `WorldNormal`（ライティング用）  
  - `uv`（テクスチャ用）

→ **Static / Skinned 両対応の統合 VS** になっています。

---

### 2. マテリアル定数バッファ（MaterialCB）

- `usePBR`：0 = Phong, 1 = PBR  
- `useTexture`：0 = 色のみ、1 = テクスチャ使用  
- `baseColor`：共通のベースカラー  
- Phong 用：`diffuseColor`, `specularColor`, `shininess`  
- PBR 用：`metallic`, `roughness`  

→ **CPU 側（C++）からモードを切り替えるだけで Phong / PBR を選べる**。

---

### 3. ピクセルシェーダ

- ライト方向・カメラ位置は PerFrameCB から  
- `usePBR` で分岐：
  - 0 → Phong 風 BRDF  
  - 1 → 簡易 PBR BRDF  
- `useTexture` でテクスチャの有無を切り替え  

→ **統合シェーダー 1 本で Phong / PBR / Static / Skinned を全部カバー**。

## 🧩 いまの Material が持っているもの  
あなたの Material は、Assimp から読み込んだ **テクスチャのパス情報だけ**を保持しています。

例：

```cpp
struct TextureInfo {
    std::string path;
    int uvIndex = 0;
};

class Material {
public:
    TextureInfo baseColorTex;
    TextureInfo metallicRoughnessTex;
    TextureInfo normalTex;
    TextureInfo emissiveTex;
    TextureInfo occlusionTex;
    TextureInfo diffuseTex;
    TextureInfo specularTex;
};
```

Assimp の `aiMaterial` から `LoadTex()` でパスを読み込んでいるので、  
**Blender → glTF → Assimp のテクスチャパスは正しく Material に入っています。**

---

## 🎯 しかし「Texture クラス」が無いので GPU に送れていない  
DirectX11 でテクスチャを使うには：

- ID3D11ShaderResourceView（SRV）
- ID3D11Resource（テクスチャ本体）

が必要です。

しかし今の Material は **パス文字列だけ**を持っていて、  
Texture2D をロードしていません。

そのため：

- HLSL の `Texture2D baseColorTex : register(t0);` に何もバインドされていない  
- `baseColorTex.Sample()` は常に黒（0）  
- 色は ConstantBuffer の baseColor だけで決まる  
- テクスチャは一切表示されない  

という状態です。

## 🧩 テクスチャを表示するために必要なもの

### 1. Texture クラス（SRV を持つ）

```cpp
class Texture {
public:
    ComPtr<ID3D11ShaderResourceView> srv;
    ComPtr<ID3D11Resource> resource;
};
```

### 2. TextureLoader（ファイルから Texture を作る）

```cpp
std::shared_ptr<Texture> TextureLoader::Load(const std::string& path)
{
    std::wstring wpath = ConvertToWString(path);

    ComPtr<ID3D11ShaderResourceView> srv;
    HRESULT hr = CreateWICTextureFromFile(device, wpath.c_str(), nullptr, srv.GetAddressOf());
    if (FAILED(hr)) return nullptr;

    auto tex = std::make_shared<Texture>();
    tex->srv = srv;
    return tex;
}
```

### 3. Material に Texture を持たせる

```cpp
std::shared_ptr<Texture> baseColorTexture;
```

### 4. Renderer で SRV をバインドする

```cpp
context->PSSetShaderResources(0, 1, mat.baseColorTexture->srv.GetAddressOf());
```
